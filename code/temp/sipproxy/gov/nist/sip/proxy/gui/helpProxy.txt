
You can parameterize the proxy via the configuration menu. 
You have to take a look the docs/gov/nist/sip/proxy/README.html for a
detailed explanation of the parameters.

-------------------------------------------------------------------------------
IMPORTANT:
    - You have to restart (stop/start button) the proxy once you applied the changes
for the parameters.
