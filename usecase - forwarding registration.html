Παράδειγμα Περίπτωσης Χρήσης: Καταχώριση Προώθησης

  • Δράστες: προωθών, αποδέκτης, client προωθόντα, forwarding server, βάση δεδομένων

  • Προαπαιτούμενα (Preconditions):

    • Ο προωθών είναι ενεργός.
    • Ο forwarding server είναι ενεργός.

  • Βασική Πορεία Χρήσης:

    1. O προωθών πληκτρολογεί το όνομα χρήστη του αποδέκτη και επιλεγει τη προώθηση.
    2. Ο forwarding server λαμβάνει το αίτημα για προώθηση από τον client του προωθόντα.
    3. Ο forwarding server αναζητεί στη βάση δεδομένων χρήστη με το όνομα χρήστη του αποδέκτη.
    4. [TODO]
    5. Ο forwarding server ενημερώνει τη βάση δεδομένων, θέτοντας τον αποδέκτη ως διεύθυνση προώθησης του προωθόντα.
    6. Ο forwarding server αποστέλει επιβεβαίωση επιτυχούς προώθησης στον client του προωθόντα.
    7. Ο client του προωθόντα εμφανίζει κατάλληλο μύνημα επιτυχίας.

  • Εναλλακτικές πορείες χρήσης:

    • Εάν δεν υπάρχει χρήστης με το όνομα χρήστη του αποδέκτη στη βάση δεδομένων:

      • Ο forwarding server παράγει μύνημα αποτυχίας προώθησης.
      • Ο forwarding server αποστέλει το μήνυμα στον client του προωθόντα.
      • Ο client του προωθόντα εμφανίζει το σχετικό μύνημα.
