Παράδειγμα Περίπτωσης Χρήσης: Καταχώριση Προώθησης

  • Δράστες: προωθών, forwarding server, βάση δεδομένων, proxy server

  • Προαπαιτούμενα (Preconditions):

    • Ο προωθών είναι ενεργός.
    • Ο forwarding server είναι ενεργός.
    • Ο proxy server είναι ενεργός.

  • Βασική Πορεία Χρήσης:

    1. O προωθών επιλέγει προώθηση κλήσης προς άλλον χρήστη.
    2. Ο proxy server λαμβάνει το αίτημα για προώθηση από τον προωθόντα.
    3. Ο forwarding server λαμβάνει το αίτημα για προώθηση από τον proxy server.
    4. Ο forwarding server αναζητεί στη βάση δεδομένων χρήστη με το όνομα χρήστη του αποδέκτη κλήσεων.
    5. Ο forwarding server επιβεβαιώνει τη μη δημιουργία κύκλου στην αλυσίδα προώθησης μεταξύ προωθόντα και αποδέκτη, αναζητώντας τις προωθήσεις στη βάση δεδομένων.
    6. Ο forwarding server ενημερώνει τη βάση δεδομένων, θέτοντας τον αποδέκτη ως διεύθυνση προώθησης του προωθόντα.
    7. Ο forwarding server αποστέλει επιβεβαίωση επιτυχούς προώθησης στον proxy server.
    8. Ο proxy server αποστέλει επιβεβαίωση επιτυχούς προώθησης στον προωθόντα.

  • Εναλλακτικές πορείες χρήσης:

    • Εάν δεν υπάρχει χρήστης με το όνομα χρήστη του αποδέκτη στη βάση δεδομένων:

      • Ο forwarding server αποστέλει μήνυμα αποτυχίας προώθησης στον proxy server.
      • Ο proxy server αποστέλει μήνυμα αποτυχίας προώθησης στον προωθόντα.

    • Εάν δημιουργείται κύκλος στην αλυσίδα προώθησης:

      • Ο forwarding server αποστέλει μήνυμα αποτυχίας προώθησης στον proxy server.
      • Ο proxy server αποστέλει μήνυμα αποτυχίας προώθησης στον προωθόντα.
