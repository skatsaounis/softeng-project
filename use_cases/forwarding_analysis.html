Παράδειγμα Περίπτωσης Χρήσης: Επίλυση Προώθησης

  • Δράστες: forwarding server, proxy server, blocking server, βάση δεδομένων

  • Προαπαιτούμενα (Preconditions):

    • Ο proxy server είναι ενεργός.
    • Ο forwarding server είναι ενεργός.
    • Ο blocking server είναι ενεργός.
    • Ο forwarding server επιλύει τη διαδρομή της κλήσης και εντοπίζει τον τελικό αποδέκτη.

  • Βασική Πορεία Χρήσης:

    1. Ο proxy server ενημερώνει τον forwarding server να πραγματοποιήσει επίλυση διαδρομής κλήσης.
    2. Ο forwarding server εντοπίζει από τη βάση δεδομένων όλους τους ενδιάμεσους χρήστες της αλυσίδας προώθησης.
    3. Ο forwarding server επιβεβαιώνει μέσω του blocking server ότι κανένας από τους ενδιάμεσους χρήστες δεν φράσσει κάποιον από τους προηγούμενούς του στην αλυσίδα προώθησης. Επιπλέον επιβεβαιώνει μέσω του blocking server ότι κανένας από τους ενδιάμεσους χρήστες δεν φράσσει τον καλούντα.
    4. Ο forwarding server επιβεβαιώνει ότι ο τελικός αποδέκτης δε συμπίπτει με τον καλούντα.
    5. Ο forwarding server ενημερώνει τον proxy server για τον τελικό αποδέκτη της κλήσης.

  • Εναλλακτικές πορείες χρήσης:

    • Εάν κάποιος από τους ενδιάμεσους χρήστες φράσσει κάποιον από τους προηγούμενούς του στην αλυσίδα προώθησης ή κάποιος από τους ενδιάμεσους χρήστες φράσσει τον καλούντα:

      • Ο forwarding server αποτυγχάνει να βρει τον τελικό αποδέκτη και αποστέλει μήνυμα απασχολημένου στον proxy server.

    • Εάν ο τελικός αποδέκτης συμπίπτει με τον καλούντα:

      • Ο forwarding server αποτυγχάνει να βρει τον τελικό αποδέκτη και αποστέλει μήνυμα απασχολημένου στον proxy server.
