Παράδειγμα Περίπτωσης Χρήσης: Αφαίρεση Φραγής

  • Δράστες: φράσσων, blocking server, βάση δεδομένων, proxy server

  • Προαπαιτούμενα (Preconditions):

    • Ο φράσσων είναι ενεργός.
    • Ο proxy server είναι ενεργός.
    • Ο blocking server είναι ενεργός.

  • Βασική Πορεία Χρήσης:

    1. O φράσσων επιλεγει αφαίρεση φραγής κλήσεων από τον φρασσόμενο.
    2. Ο proxy server λαμβάνει το αίτημα για αφαίρεση φραγής από τον φράσσοντα.
    3. Ο proxy server προωθεί το αίτημα στον blocking server.
    4. Ο blocking server αναζητεί στη βάση δεδομένων χρήστη με το όνομα χρήστη του φρασσόμενου.
    5. Ο blocking server επιβεβαιώνει ότι ο φρασσόμενος βρίσκεται στις φραγμένες επαφές του φράσσοντα, αναζητώντας τη βάση δεδομένων.
    6. Ο blocking server ενημερώνει τη βάση δεδομένων, αφαιρώντας τον φρασσόμενο από τις φραγμένες επαφές του φράσσοντα.
    7. Ο blocking server αποστέλλει επιβεβαίωση αφαίρεσης φραγής στον proxy server.
    8. Ο proxy server αποστέλλει επιβεβαίωση αφαίρεσης φραγής στον φράσσοντα.

  • Εναλλακτικές πορείες χρήσης:

    • Εάν δεν υπάρχει χρήστης με το όνομα χρήστη του φρασσόμενου στη βάση δεδομένων:

      • Ο blocking server αποστέλλει μήνυμα αποτυχίας αφαίρεσης φραγής στον proxy server.
      • Ο proxy server αποστέλλει μήνυμα αποτυχίας αφαίρεσης φραγής στον φράσσοντα.

    • Εάν ο φρασσόμενος δεν βρίσκεται στις φραγμένες επαφές του φράσσοντα:

      • Ο blocking server αποστέλλει επιβεβαίωση αφαίρεσης φραγής στον proxy server.
      • Ο proxy server αποστέλλει επιβεβαίωση αφαίρεσης φραγής στον φράσσοντα.
