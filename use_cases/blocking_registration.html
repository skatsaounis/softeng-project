Παράδειγμα Περίπτωσης Χρήσης: Καταχώριση Φραγής

  • Δράστες: φράσσων, blocking server, βάση δεδομένων, proxy server

  • Προαπαιτούμενα (Preconditions):

    • Ο φράσσων είναι ενεργός.
    • Ο proxy server είναι ενεργός.
    • Ο blocking server είναι ενεργός.

  • Βασική Πορεία Χρήσης:

    1. O φράσσων επιλεγει φραγή κλήσεων από τον φρασσόμενο.
    2. Ο proxy server λαμβάνει το αίτημα για φραγή από τον φράσσοντα.
    3. Ο proxy server προωθεί το αίτημα στον blocking server.
    4. Ο blocking server αναζητεί στη βάση δεδομένων χρήστη με το όνομα χρήστη του φρασσόμενου.
    5. Ο blocking server επιβεβαιώνει ότι ο φρασσόμενος δεν βρίσκεται στις φραγμένες επαφές του φράσσοντα, αναζητώντας τη βάση δεδομένων.
    6. Ο blocking server ενημερώνει τη βάση δεδομένων, προσθέτοντας τον φρασσόμενο στις φραγμένες επαφές του φράσσοντα.
    7. Ο blocking server αποστέλει επιβεβαίωση επιτυχούς φραγής στον proxy server.
    8. Ο proxy server αποστέλει επιβεβαίωση επιτυχούς φραγής στον φράσσοντα.

  • Εναλλακτικές πορείες χρήσης:

    • Εάν δεν υπάρχει χρήστης με το όνομα χρήστη του φρασσόμενου στη βάση δεδομένων:

      • Ο blocking server αποστέλει μήνυμα αποτυχίας φραγής στον proxy server.
      • Ο proxy server αποστέλει μήνυμα αποτυχίας φραγής στον φράσσοντα.

    • Εάν ο φρασσόμενος βρίσκεται ήδη στις φραγμένες επαφές του φράσσοντα:

      • Ο blocking server αποστέλει επιβεβαίωση επιτυχούς φραγής στον proxy server.
      • Ο proxy server αποστέλει επιβεβαίωση επιτυχούς φραγής στον φράσσοντα.
