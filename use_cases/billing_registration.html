Παράδειγμα Περίπτωσης Χρήσης: Καταχώριση Προγράμματος

  • Δράστες: χρήστης, billing server, βάση δεδομένων, proxy server

  • Προαπαιτούμενα (Preconditions):

    • Ο χρήστης είναι ενεργός.
    • Ο proxy server είναι ενεργός.
    • Ο billing server είναι ενεργός.

  • Βασική Πορεία Χρήσης:

    1. O χρήστης επιλέγει πρόγραμμα χρέωσης.
    2. Ο proxy server λαμβάνει το αίτημα για επιλογή προγράμματος χρέωσης από τον χρήστη.
    3. Ο proxy server προωθεί το αίτημα στον billing server.
    4. Ο billing server αναζητεί στη βάση δεδομένων το όνομα του χρήστη.
    5. Ο billing server επιβεβαιώνει ότι ο χρήστης δεν έχει επιλέξει πρόγραμμα τα τελευταία 5 λεπτά, αναζητώντας τη βάση δεδομένων.
    6. Ο billing server ενημερώνει τη βάση δεδομένων, αλλάζοντας το πρόγραμμα χρέωσης του χρήστη.
    7. Ο billing server αποστέλλει επιβεβαίωση επιτυχούς αλλαγής προγράμματος χρέωσης στον proxy server.
    8. Ο proxy server αποστέλλει επιβεβαίωση επιτυχούς αλλαγής προγράμματος χρέωσης στον χρήστη.

  • Εναλλακτικές πορείες χρήσης:

    • Εάν ο χρήστης έχει επιλέξει πρόγραμμα χρέωσης έως και 5 λεπτά νωρίτερα:

      • Ο billing server αποστέλλει μήνυμα αποτυχίας αλλαγής προγράμματος χρέωσης στον proxy server.
      • Ο proxy server αποστέλλει μήνυμα αποτυχίας αλλαγής προγράμματος χρέωσης στον χρήστη.
