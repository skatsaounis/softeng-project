Παράδειγμα Περίπτωσης Χρήσης: Αναφορά συνολικής χρέωσης

  • Δράστες: χρήστης, billing server, βάση δεδομένων, proxy server

  • Προαπαιτούμενα (Preconditions):

    • Ο χρήστης είναι ενεργός.
    • Ο proxy server είναι ενεργός.
    • Ο billing server είναι ενεργός.

  • Βασική Πορεία Χρήσης:

    1. Ο χρήστης πραγματοποιεί αίτημα αναφοράς της συνολικής χρέωσης του.
    2. O proxy server ζητά από τον billing server να υπολογίσει τη συνολική χρέωση του χρήστη.
    3. Ο billing server αναζητεί τον χρήστη στη βάση δεδομένων και τις κλήσεις που έχει πραγματοποιήσει.
    4. Ο billing server υπολογίζει τη συνολική χρέωση του χρήστη.
    5. Ο billing server αποστέλλει στον proxy server τη συνολική χρέωση του χρήστη.
    6. Ο proxy server αποστέλλει στον χρήστη τη συνολική του χρέωση.
