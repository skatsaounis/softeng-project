Παράδειγμα Περίπτωσης Χρήσης: Χρέωση κλήσης

  • Δράστες: billing server, βάση δεδομένων, proxy server

  • Προαπαιτούμενα (Preconditions):

    • Έχει γίνει έναρξη κλήσης.
    • Ο billing server είναι ενεργός.

  • Βασική Πορεία Χρήσης:

    1. O proxy server ενημερώνει τον billing server ότι εκκίνησε η κλήση ανάμεσα στον καλούντα και τον καλούμενο.
    2. Ο billing server αναζητεί τον καλούντα στη βάση δεδομένων.
    3. Ο billing server καταγράφει στη βάση δεδομένων την έναρξη της κλήσης από τον καλούντα στον καλούμενο, καθώς και το πρόγραμμα χρέωσης του καλούντα.
    4. Ο proxy server ενημερώνει τον billing server ότι τερματίστηκε η κλήση ανάμεσα στον καλούντα και τον καλούμενο.
    5. Ο billing server καταγράφει στη βάση δεδομένων τον τερματισμό της κλήσης από τον καλούντα στον καλούμενο.
