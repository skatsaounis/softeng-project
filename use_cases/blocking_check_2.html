Παράδειγμα Περίπτωσης Χρήσης: Έλεγχος Φραγής

  • Δράστες: blocking server, forwarding server, βάση δεδομένων

  • Προαπαιτούμενα (Preconditions):

    • Ο forwarding server είναι ενεργός.
    • Ο blocking server είναι ενεργός.
    • Ο forwarding server ελέγχει για φραγή κλήσεων.
    • Ο forwarding server αποστέλει στον blocking server την αλυσίδα forwarding και το όνομα χρήστη του καλούντα.

  • Βασική Πορεία Χρήσης:

    1. Ο forwarding server ενημερώνει τον blocking server να πραγματοποιήσει έλεγχο για φραγή.
    2. Ο blocking server επιβεβαιώνει ότι ο καλών δεν φράσσεται από κάποιον ενδιάμεσο χρήστη της αλυσίδας forwarding, αναζητώντας τη βάση δεδομένων.
    3. Ο blocking server επιβεβαιώνει ότι κανείς από τους ενδιάμεσους χρήστες της αλυσίδας forwarding δεν φράσσει κάποιον από τους προηγούμενούς του.
    4. Ο blocking server αποστέλει μήνυμα επιτρεπτής κλήσης στον forwarding server.

  • Εναλλακτικές πορείες χρήσης:

    • Εάν ο καλών φράσσεται από κάποιον ενδιάμεσο χρήστη της αλυσίδας forwarding:

      • Ο blocking server αποστέλει μήνυμα φραγής στον forwarding server.

    • Εάν κάποιος από τους ενδιάμεσους χρήστες της αλυσίδας forwarding φράσσει κάποιον από τους προηγούμενούς του:

      • Ο blocking server αποστέλει μήνυμα φραγής στον forwarding server.
