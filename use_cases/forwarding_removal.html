Παράδειγμα Περίπτωσης Χρήσης: Αφαίρεση Προώθησης

  • Δράστες: προωθών, forwarding server, βάση δεδομένων, proxy server

  • Προαπαιτούμενα (Preconditions):

    • Ο προωθών είναι ενεργός.
    • Ο forwarding server είναι ενεργός.
    • Ο proxy server είναι ενεργός.

  • Βασική Πορεία Χρήσης:

    1. O προωθών επιλέγει την αφαίρεση προώθησης.
    2. Ο proxy server λαμβάνει το αίτημα για αφαίρεση προώθησης από τον προωθόντα.
    3. Ο forwarding server λαμβάνει το αίτημα για αφαίρεση προώθησης από τον proxy server.
    4. Ο forwarding server αναζητεί στη βάση δεδομένων τον προωθόντα.
    5. Ο forwarding server ενημερώνει τη βάση δεδομένων, αφαιρώντας τη διεύθυνση προώθησης του προωθόντα.
    6. Ο forwarding server αποστέλει επιβεβαίωση αφαίρεσης προώθησης στον proxy server.
    7. Ο proxy server αποστέλει επιβεβαίωση αφαίρεσης προώθησης στον προωθόντα.

  • Εναλλακτικές πορείες χρήσης:

    • Εάν δεν υπάρχει διεύθυνση προώθησης κλήσεων του προωθόντα στη βάση δεδομένων:

      • Ο forwarding server αποστέλει επιβεβαίωση αφαίρεσης προώθησης στον proxy server.
      • Ο proxy server αποστέλει επιβεβαίωση αφαίρεσης προώθησης στον προωθόντα.
