Παράδειγμα Περίπτωσης Χρήσης: Έλεγχος Φραγής

  • Δράστες: blocking server, proxy server, βάση δεδομένων

  • Προαπαιτούμενα (Preconditions):

    • Ο proxy server είναι ενεργός.
    • Ο blocking server είναι ενεργός.
    • Ο proxy server ελέγχει για φραγή κλήσεων.

  • Βασική Πορεία Χρήσης:

    1. Ο proxy server ενημερώνει τον blocking server να πραγματοποιήσει έλεγχο για φραγή.
    2. Ο blocking server επιβεβαιώνει ότι ο καλών βρίσκεται στις φραγμένες επαφές του καλούμενου, αναζητώντας τη βάση δεδομένων.
    3. Ο blocking server ενημερώνει τον proxy server ότι ο καλούμενος έχει ενεργοποιήσει φραγή κλήσεων για τον καλούντα.

  • Εναλλακτικές πορείες χρήσης:

    • Εάν ο καλών δεν βρίσκεται στις φραγμένες επαφές του καλούμενου:

      • Ο blocking server αποστέλει μήνυμα επιτρεπτής κλήσης στον proxy server.
