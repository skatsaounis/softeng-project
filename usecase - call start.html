Παράδειγμα Περίπτωσης Χρήσης: Έναρξη Κλήσης

  • Δράστες: καλών, καλούμενος, proxy server, βάση δεδομένων

  • Προαπαιτούμενα (Preconditions):

    • Ο proxy server είναι ενεργός.
    • Ο καλών έχει καταχωριστεί στη βάση δεδομένων του proxy server.
    • Ο καλούμενος έχει καταχωριστεί στη βάση δεδομένων του proxy server.
    • Ο καλών είναι ενεργός.

  • Βασική Πορεία Χρήσης:

    1. O καλών επιχειρεί πραγματοποίηση κλήσης για τον καλούμενο.
    2. Ο proxy server λαμβάνει το αίτημα του καλούντος.
    3. Ο proxy server ελέγχει ότι το όνομα χρήστη του καλούμενου είναι διαφορετικό από του καλούντος.
    4. Ο proxy server αναζητάει στη βάση δεδομένων χρήστη με αυτό το όνομα χρήστη.
    5. Ο proxy server επιβεβαιώνει τη μη φραγή κλήσεων.
    6. Ο proxy server αναζητάει στη βάση δεδομένων αν είναι ενεργός ο χρήστης - καλούμενος.
    7. Ο proxy server αναζητάει στη βάση δεδομένων αν είναι απασχολημένος ο χρήστης - καλούμενος.
    8. Ο proxy server προωθεί το αίτημα κλήσης στον καλούμενο.
    9. Ο καλούμενος αποδέχεται την κλήση.
    10. Η κλήση εκκινεί.

  • Εναλλακτικές πορείες χρήσης:

    • Εάν το όνομα χρήστη του καλούμενου είναι ίδιο με του καλούντος:

      • Ο proxy server παράγει μήνυμα απασχολημένου.
      • Ο proxy server αποστέλει το μήνυμα στον καλούντα.

    • Εάν δεν υπάρχει χρήστης με το όνομα χρήστη του καλούμενου στη βάση δεδομένων:

      • Ο proxy server παράγει μήνυμα περί μη εύρεσης χρήστη στη βάση δεδομένων.
      • Ο proxy server αποστέλει το μήνυμα στον καλούντα.

    • Ο καλούμενος έχει ενεργοποιήσει φραγή κλήσεων στον καλούντα:

      • Ο proxy server παράγει μήνυμα απασχολημένου.
      • Ο proxy server αποστέλει το μήνυμα στον καλούντα.

    • Εάν δεν είναι ενεργός χρήστης ο καλούμενος:

      • Ο proxy server παράγει μήνυμα ανενεργού χρήστη.
      • Ο proxy server αποστέλει το μήνυμα στον καλούντα.

    • Εάν ο καλούμενος είναι απασχολημένος:

      • Ο proxy server παράγει μήνυμα απασχολημένου.
      • Ο proxy server αποστέλει το μήνυμα στον καλούντα.

    • Εάν ο καλούμενος απορρίψει την κλήση:

      • Ο proxy server παράγει μήνυμα απασχολημένου.
      • Ο proxy server αποστέλει το μήνυμα στον καλούντα.
